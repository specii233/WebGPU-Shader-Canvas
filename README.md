# WebGPU-Shader-Canvas
A simple canvas effect with WebGPU shader

&gt; 最小可运行 **WebGPU** 示例 —— 无打包器，裸跑 `index.html`，浏览器里直接画色块，完整走完 **device → pipeline → draw** 全程。
